# Getting Started

This section walks you through installing Charlie UI, setting up your project, and building your first interface with the library.

## What You Will Learn

- **[Installation](installation.md)** -- How to add Charlie UI to your project, including peer dependencies and CSS setup.
- **[Quick Start](quick-start.md)** -- A hands-on guide to building a simple page layout using core components.
- **[Design Tokens](design-tokens.md)** -- The color palette, typography, spacing scale, radii, and shadows that power the design system.

## Prerequisites

Charlie UI is built for React 18 or 19 projects using TypeScript. You will also need a working Tailwind CSS v4 setup. If you are starting a new project, we recommend using Vite with the `@vitejs/plugin-react` and `@tailwindcss/vite` plugins.
